RSA Cryptosystem
Github

A cryptosystem which uses the RSA algorithm to generate public and private key pairs, uses the public key to encrypt files, and the corresponding private key to decrypt those encrypted files.


Both the encryptor and decryptor have the ability to read from/write to stdin and stdout but can also use files specified by command-line arguments when running the program.

Data Compressor
Github

A lossless data compressor and decompressor using Huffman coding.

A priority queue (maintained by a min heap) is used to construct the Huffman tree and a stack is used to rebuild the tree from the encoded output.
